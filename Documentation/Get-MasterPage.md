# Get-MasterPage

## SYNOPSIS
Returns the URLs of the default Master Page and the custom Master Page.

## SYNTAX 

### 
```powershell
Get-MasterPage [-Context <SPOnlineContext>]
```

## PARAMETERS

### -Context


```yaml
Type: SPOnlineContext
Parameter Sets: 

Required: False
Position: 0
Accept pipeline input: False
```
