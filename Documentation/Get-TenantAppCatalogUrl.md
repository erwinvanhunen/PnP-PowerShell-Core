# Get-TenantAppCatalogUrl

## SYNOPSIS
Retrieves the url of the tenant scoped app catalog.

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
PS:> Get-PnPTenantAppCatalogUrl
```

Returns the url of the tenant scoped app catalog site collection
