# Get-Web

## SYNOPSIS
Returns the current web object

## SYNTAX 

### 
```powershell
Get-Web [-Context <SPOnlineContext>]
```

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
PS:> Get-PnPWeb
```

This will return the current web

## PARAMETERS

### -Context


```yaml
Type: SPOnlineContext
Parameter Sets: 

Required: False
Position: 0
Accept pipeline input: False
```
