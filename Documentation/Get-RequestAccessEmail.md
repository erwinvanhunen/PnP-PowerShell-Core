# Get-RequestAccessEmail

## SYNOPSIS
Returns the request access e-mail addresses

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
PS:> Get-PnPRequestAccessEmails
```

This will return all the request access e-mail addresses for the current web
