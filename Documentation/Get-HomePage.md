# Get-HomePage

## SYNOPSIS
Returns the URL to the page set as home page

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
PS:> Get-PnPHomePage
```

Will return the URL of the home page of the web.

## OUTPUTS

### System.String
