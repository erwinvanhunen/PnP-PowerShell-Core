# Get-HomePage

## SYNOPSIS
Returns the URL to the page set as home page

## SYNTAX 

### 
```powershell
Get-HomePage [-Context <SPOnlineContext>]
```

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
PS:> Get-PnPHomePage
```

Will return the URL of the home page of the web.

## PARAMETERS

### -Context


```yaml
Type: SPOnlineContext
Parameter Sets: 

Required: False
Position: 0
Accept pipeline input: False
```

## OUTPUTS

### System.String
